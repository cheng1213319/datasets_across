# Unified Heterogeneity-Aware Benchmark of Drug Synergy Prediction

## Requirements
- python=3.8
- pytorch=1.13.1
- sklearn=1.3.0
- pandas=2.0.3
- xgboost=2.0.3

## Start
Run the dataexam_5fold.py to train the model.

## Data
The training/validation/testing data is stored in the `data` folder.
The fearture files are in the `dataset_1` folder.

## Source code of the SOTA methods
- DeepDDS: https://github.com/Sinwang404/DeepDDS/tree/master
- MTLSynergy: https://github.com/TOJSSE-iData/MTLSynergy
- CGMS: https://github.com/TOJSSE-iData/CGMS
- JointSyn: https://github.com/LiHongCSBLab/JointSyn
