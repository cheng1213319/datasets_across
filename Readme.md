# Unified Heterogeneity-Aware Benchmark of Drug Synergy Prediction

## Requirements

Python 3.8 is required. It is strongly recommended to use a virtual environment.

Install all dependencies with:
```bash
pip install -r requirements.txt
```
The `requirements.txt` file lists all necessary packages with pinned versions to ensure reproducibility.

## Start
Run the `dataexam_5fold.py` to train the model.
Detailed usage instructions (including how to specify dataset, model, and synergy metric) are provided as comments in the script.

## Data
The training/validation/testing data is stored in the `data` folder.
The fearture files are in the `dataset_1` folder.

## Reproducing figures and tables
All figures and tables in the paper can be generated by running `jupyter notebooks` file in `visual1/`
